# Python: de Fundamentos a Aplicações com LLMs

Repositório dedicado ao estudo contínuo de Python, cobrindo desde a sintaxe essencial e lógica de programação até o desenvolvimento de APIs robustas e a integração avançada com Modelos de Linguagem de Larga Escala (LLMs).

## 🚀 Tecnologias e Ecossistema

O repositório está estruturado em três pilares principais:

* **Fundamentos de Python:** Estruturas de dados, orientação a objetos, manipulação de ficheiros e boas práticas de código.
* **Frameworks Web & APIs:** Criação de serviços assíncronos e modulares.
    * **Django:** Arquitetura MVC/MVT, ORM, painel administrativo e ecossistema robusto.
    * **FastAPI:** Alta performance, tipagem estática (Pydantic), validação automática e documentação interativa nativa (Swagger).
* **Engenharia de Prompt & Inteligência Artificial:**
    * **LangChain:** Orquestração de pipelines e correntes (*chains*) de IA.
    * **Integração de LLMs:** Conexão, consumo e parametrização de APIs de modelos generativos (como OpenAI e Gemini).
